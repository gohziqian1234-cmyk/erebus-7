# Recorded Voice Pack

Add your own recorded voiceover clips here. The game will use these files before falling back to browser speech.

Important: record original lines yourself, or use a voice actor/voice you have permission to use. Do not clone a real person's voice.

Supported extensions:

- `.mp3`
- `.wav`
- `.ogg`
- `.m4a`
- `.mp4`

## Opening Story

Name the five opening transmission clips:

- `story-01.mp3`
- `story-02.mp3`
- `story-03.mp3`
- `story-04.mp3`
- `story-05.mp3`

## Prologue Scene

- `prologue-01.mp3`
- `prologue-02.mp3`
- `prologue-03.mp3`
- `prologue-04.mp3`
- `prologue-05.mp3`

## Chapter Scenes

Use this pattern:

```text
chapter-01-01.mp3
chapter-01-02.mp3
chapter-01-03.mp3
chapter-02-01.mp3
...
chapter-07-03.mp3
```

## Blackbox Logs

Use this pattern:

```text
blackbox-01-01.mp3
blackbox-01-02.mp3
blackbox-01-03.mp3
...
blackbox-07-03.mp3
```

For the style, aim for close-mic horror radio: calm, low volume, slow pacing, slight breath, no music baked into the clip.
